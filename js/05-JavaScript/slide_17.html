Arrays

 

  • Arrays are a special kind of objects, with numbered indexes.
    • objects create a map between keys and values
      • The key can be integer or everything else
    • Arrays with integer index (key)
      • var a = [1, 2, 3];
      const a = new Array (10, 20, ...);
      const a = new Array();   a[10] = "HTML"; a[120] = "JS";
    • Objects
      • var style = {color: "Blue", size: "20pt"};
      var c = style["color"]; var c = style.color;
  • Methods: concat, shift, unshift, sort, reverse, indexOf, ...
