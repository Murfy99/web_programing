JavaScript Syntax

 

  • Syntax is very similar to C/Java/C++ (But with some differences)
  • Semicolon is required between multiple statements in a line
    • If a single statement in line, semicolon is optional
  • There is not any main function
  • Variables are declared by var, let, const (is optional)
    • Required for local variables (inside functions)
      • without declaration, variables inside the functions are GLOBAL
  • Functions are defined by function
